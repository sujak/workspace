# OS 재설치 후 설정

macOS High Sierra 10.13.4 기준으로 작성

### 컴퓨터 이름 변경

- 시스템 환경설정 -> 공유 -> 컴퓨터이름
- `sudo scutil --set HostName 새로운호스트명`

### 키보드 설정 변경

- 맞춤법 자동 수정 X
- 자동으로 문장을 대문자로 시작 X
- 스페이스 두번 마침표 X

### 기타 설정

- 알림센터 편집
- 배터리 퍼센트 보기
